    fullcalendar_rails

    Using FullCalendar jQuery plugin(arshaw.com/fullcalendar/) with Rails backend. You can create, edit, delete, reschedule, resize events.

    Rails Engine implementation is also available for this functionality.

    Rails fullcalendar engine gem: rubygems.org/gems/fullcalendar_engine

    Rails fullcalendar engine source: github.com/vinsol/fullcalendar-rails-engine

    Rails fullcalendar engine demo implentation: vinsol.com/fullcalendar-demo
